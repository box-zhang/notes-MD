# VS Code 插件使用

## Prettier - 格式化插件

快捷键：`cmd + shift + f`

## fileheader - 文件注释

快捷键：`ctrl + alt + i`

在settings.json里设置：

~~~json
{
    "fileheader.Author":"boxZhang",
    "fileheader.LastModifiedBy":"boxZhang",
}
~~~

## Document This - 文件注释

快捷键：`ctrl + alt + D` 连续按两遍

只支持JavaScript和TypeScript
